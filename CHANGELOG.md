## v0.2.2

> 부스 목록 — 장르 필터 다중 선택

### 개선
- **장르 필터 다중 선택**: 여러 장르를 동시에 선택할 수 있습니다. 2개 이상 선택 시 하나라도 포함된 부스가 모두 표시됩니다.
- 선택된 장르 수와 태그만 초기화하는 버튼이 별도로 표시됩니다.
- **필터 초기화 버튼 항상 표시**: 이전에는 필터가 적용된 경우에만 나타나 레이아웃이 흔들리는 문제가 있었습니다. 항상 표시되며, 필터 적용 중일 때 강조됩니다.

## v0.2.1

> 부스 목록 — 요일 필터 개선

### 개선
- **요일 필터 방식 변경**: 기존에는 토요일 선택 시 토요일이 포함된 부스(양일 참가 포함)가 모두 표시됐습니다. 이제 해당 요일만 단독 참가하는 부스만 표시합니다.
- 요일 버튼이 실제 부스 데이터를 기반으로 동적 생성됩니다. 토/일 행사뿐 아니라 금/토/일 등 다양한 일정의 행사에 자동으로 대응합니다.

## v0.1.0 — 최초 릴리즈

> 사이트 오픈

### 신규 페이지

**메인 페이지**
- 진행 중인 행사가 있으면 우선 표시, 없으면 이번 달 예정 행사, 그것도 없으면 가장 가까운 행사 3개를 자동으로 보여줍니다.
- 행사·부스·작가·작품 목록 바로가기 제공.

**행사 목록** (`/events`)
- 코믹월드, 일러스타 페스, 코미버스 행사 목록.
- 행사 타입 / 도시 / 상태별 필터.
- 지난 행사는 기본적으로 2년 이내만 표시하며, 전체 표시로 전환 가능.
- 진행 중인 행사는 초록 점 애니메이션으로 강조.

**행사 상세** (`/events/:id`)
- 일정, 장소, 협력 행사, 공식 링크(홈페이지/Twitter/Bluesky/DC갤러리) 표시.
- 카카오맵 검색 링크로 장소 확인 가능.
- 부스 목록 바로가기 버튼.

**부스 목록** (`/booths/:eventId`)
- 카드형 / 리스트형 뷰 전환.
- 부스명·번호·태그 검색, 규격·요일 필터, 장르 태그 필터.
- 부스 클릭 시 상세 모달 (이미지, 링크, 취급 장르).

**작가 목록** (`/artists`)
- 작가 프로필 카드: 닉네임, SNS 링크, 주요 장르, 참여 부스 수.
- 이름 검색, 플랫폼 필터, 장르 태그 필터.
- 작가 클릭 시 상세 모달: 링크트리 스타일 SNS 링크, 참여 부스 목록.

**작품/장르 목록** (`/works`)
- 가나다 초성 기준 그룹핑 및 정렬.
- 초성 인덱스 클릭 시 해당 섹션으로 이동.
- 각 작품에 부스 참여 횟수 배지 표시.

**이벤트 캘린더** (`/calendar`)
- 월간 뷰: 달력에 행사 일정 표시, 클릭하면 행사 상세로 이동.
- 주간 뷰: 타임라인 바 형식, 같은 날 겹치는 행사는 자동으로 레인 분리.

**통계** (`/stats`)
- 부스 탭: 규격 분포, 요일 참가 현황, 부스 크기 분포, 장르 Top 20.
- 행사 탭: 연도별·월별 추이, 타입·도시별 분포.

**기여자** (`/contributors`)
- 개발자·기여자 프로필 및 사용 기술 목록.
